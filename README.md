# Jagalapak - Remote Digital Enabler

Jagalapak adalah solusi **"Virtual Digital Manager"** bagi UMKM konvensional, khususnya di wilayah Sulawesi Selatan. Proyek ini bertujuan untuk membantu toko konvensional bertransformasi menjadi toko modern melalui pengelolaan operasional secara jarak jauh (remote).
